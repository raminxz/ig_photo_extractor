# استخراج‌کننده عکس‌های اینستاگرام

اپلیکیشن فلسک برای دانلود تمام **عکس‌های** یک پیج عمومی اینستاگرام (بدون ویدیو)، با نمایش پیشرفت زنده و قابلیت مکث/ادامه.

## نصب

```bash
cd ig_photo_extractor
pip install -r requirements.txt
```

## اجرا

```bash
python app.py
```

سپس در مرورگر به آدرس `http://localhost:5000` برید.

## استفاده

1. نام کاربری پیج مورد نظر رو وارد کنید (بدون @) — فقط پیج‌های **عمومی** جواب می‌دن.
2. اگه می‌خواید محدود به تعداد پست خاصی باشه، تو فیلد «حداکثر تعداد پست» بنویسید. خالی بذارید یعنی همه پست‌ها.
3. روی «شروع استخراج» بزنید.
4. هر وقت خواستید می‌تونید مکث/ادامه بدید یا کامل متوقف کنید.
5. وقتی کامل شد (یا حتی وسط کار)، روی «دانلود فایل zip عکس‌ها» بزنید تا همه عکس‌ها رو یکجا بگیرید.

## نکات مهم

- **فقط پیج عمومی**: این ابزار فقط با پیج‌هایی که خصوصی (private) نیستن کار می‌کنه، مگر این‌که با اکانت خودتون لاگین کنید (که در این نسخه پیاده‌سازی نشده).
- **محدودیت نرخ اینستاگرام**: بین هر دانلود یک وقفه‌ی کوتاه گذاشته شده تا احتمال بلاک شدن IP یا اکانت کم بشه. اگه با تعداد پست خیلی زیاد کار می‌کنید (چند هزار تا)، ممکنه اینستاگرام موقتاً محدودتون کنه — در این صورت چند ساعت صبر کنید و دوباره امتحان کنید (با resume ادامه از همون جا شروع می‌شه).
- **Resume**: اگه صفحه رو ببندید یا برنامه رو قطع کنید، دفعه بعد که همون یوزرنیم رو وارد کنید، عکس‌هایی که قبلاً دانلود شدن رو دوباره دانلود نمی‌کنه.
- **مسئولیت حقوقی**: قبل از دانلود محتوای هر پیج، مطمئن بشید که به قوانین اینستاگرام (Terms of Service) و حق کپی‌رایت صاحب محتوا احترام می‌ذارید. این ابزار برای استفاده شخصی، آرشیو، یا مواردی که به محتوا دسترسی/اجازه دارید مناسبه.
- فایل‌های دانلود شده تو پوشه‌ی `downloads/<username>/` ذخیره می‌شن.

## عیب‌یابی

- **خطای اتصال / rate limit**: اینستاگرام گاهی IP رو موقت محدود می‌کنه. صبر کنید و resume کنید.
- **پیج پیدا نشد**: مطمئن بشید یوزرنیم درسته و پیج عمومیه.
- **پیج خصوصیه**: این نسخه از ابزار محتوای پیج خصوصی رو دانلود نمی‌کنه.
